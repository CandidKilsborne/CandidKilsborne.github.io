DBC Pair Programming

What is pair programming?

  • A collaboration between two coders to solve a problem
  • Usually with one computer
  • Teaches one person to more hands on and the other to be more hands off

The Importance of Pair Programming

  • Improving communication
  • Increases learning
  • Leads to innovation
  • Minimizes errors

Role of Navigator

  • Directs flow of code
  • Leads communication
  • Researches topics

Role of Driver

  • Types code
  • Communicates with Navigator
  • Checks for errors
